Metodologia VRM

MC (Manual Coefficient): Preset baseado em análise de requisitos (padrão: 1.0).

SC (Standard Coefficient): SC = X_i + √(Σ(X_j - μ)² / N)

Onde: X_i = ocorrência de variável i, μ = média das ocorrências, N = número de variáveis

HC (Hierarchical Coefficient): HC = (profundidade + complexidade) × (1 / 2^profundidade).

Acoplamento (C): C = MC × SC × HC (valor sem normalização).

Interpretação:

  • Valores < 2.0: Baixo acoplamento
  • Valores 2.0 a 5.0: Acoplamento moderado
  • Valores > 5.0: Alto acoplamento
